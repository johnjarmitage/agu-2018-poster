\headerbox{Introduction}{name=introduction,column=0,row=0}{

Below the northern East African Rift two low seismic velocity zones are imaged. \textbf{Are these due to the rising thermal anomalies?} We will explore this hypothesis by generating synthetic tomography from models of the convective destabilisation of hot mantle.

%---------------------------------------------------------------------------------------
%	METHODS
%---------------------------------------------------------------------------------------

\headerbox{Geodynamics to Seismics}{name=methods,column=0,below=introduction}{

\smaller
Both Rayleigh B{\'e}nard convection and Rayleigh Taylor instabilites are generated using by solving for modified Stokes flow in a Cartesian box using the code Stag3D \cite{tackley-1998}. Temperature and pressure are converted to mineral phases assuming a pyrolite composition for the mantle and harzbugite for the lithosphere. This gives the anharmonic seismic velocity. This is corrected for attenuation using the $Q_{g}$ model \cite{goes-etal-2012}.
\begin{center}
\includegraphics[width=0.8\linewidth]{velocities}
\captionof{figure}{2D slices through the model domains for Rayleigh B{\'e}nard convection with a Newtonian rheology, a non-Newtonian rheology, and the Rayleigh Taylor destabilisation of a hot layer.}
\end{center}
}

%----------------------------------------------------------------------------------------
%	CONCLUSION
%----------------------------------------------------------------------------------------

\headerbox{To Cut a Long Story Short}{name=conclusion,column=0,below=methods}{

\begin{enumerate}
\item Only the destabilisation of a hot layer is consistent with the mangitude of the observed low seismic anomalies.
\item The spacing of the anomalies are consistent with a hot layer roughly 20,000\,km wide.
\item The magnitude of the seismic velocities are consistent a $200\rm\,^{\circ}C$ thermal anomaly.
\end{enumerate}
}

\headerbox{Rayleigh B{\'e}nard or Taylor?}{name=results1,span=2,column=1,row=0}{ % To reduce this block to 1 column width, remove 'span=2'

For Raleigh B{\'e}nard (RB) convection the wavelength between plumes, $\lambda$, varies as $Ra^{1/6}$ \cite{galsa-2007}. For Raleigh Taylor (RT) destabilisation of a hot layer, $\lambda$, varies as a linear function of the model height \cite{schmeling-1987}. The observed spacing of the two low seismic velocity structures is $\sim700$\,km (Figure 1). We find that for RB convection this implies a $10^6 < Ra 10^7$. For RT destabilisation this is consistent with a model aspect ratio of $2,1000\times2,100\times700$\,km.
